## Getting Started
```bash
$ git clone https://github.com/CDjepeno/Region_in_France-API-Symfony.git
```
```bash
$ cd Region_in_France-API-Symfony
```
```bash
$ composer install
$ npm install
```
```bash
$ Symfony server-start -d
or
$ php -S localhost:8000 -t public
```
